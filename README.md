# Timeline Coder

## Project info

A Vite + React + TypeScript application.

## How to run locally

Requirements: Bun — [install Bun](https://bun.sh/docs/installation)

```sh
# Install dependencies
bun install

# Start the development server
bun run dev
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
